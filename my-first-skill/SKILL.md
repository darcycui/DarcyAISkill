# 文本格式化器

当被要求格式化文本时:
1. 删除多余空格（将多个空格替换为单个空格）
2. 修正大小写（句子首字母大写）
3. 纠正标点符号（确保正确的结束标点）
4. 返回清理后的文本

## 示例

**输入**: "hello   world"
**输出**: "Hello world."

**输入**: "this is  a    test"
**输出**: "This is a test."

## 指南
- 保留有意的格式（换行、段落）
- 不要更改技术术语或专有名词
- 保持原始语言和语气
