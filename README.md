## 是什么

Python语言，基于 Claude Skill (Anthropic) 规则实现的一个 Skill。

## 有什么能力

格式化文本（英文）。

## 怎么用

### 在 VSCode Cline 插件中使用 skill(本地skill)

- 新建目录 C:\Users\lenovo\.cline\skills
- 把skill文件夹复制到 skills 中
- 注意 Cline 要求 SKILL.md 中的 name 必须与 skill 文件夹名称相同

### 在 VSCode Claude 插件中使用 skill(本地skill)
 - 新建目录 C:\Users\lenovo\.claude\skills
 - 把skill文件夹复制到 skills 中
